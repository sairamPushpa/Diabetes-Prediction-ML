# Diabetes Prediction using Machine Learning

## Project Overview

This project predicts whether a patient has diabetes using Machine Learning.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

## Machine Learning Algorithm

- Logistic Regression

## Dataset

Pima Indians Diabetes Dataset

## Steps

- Data Preprocessing
- Data Visualization
- Feature Scaling
- Model Training
- Prediction
- Accuracy Evaluation

## Accuracy

Approximately 80%
