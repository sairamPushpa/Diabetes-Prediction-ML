# Diabetes Prediction System Using Machine Learning

## 📌 Project Overview
This project predicts whether a patient has diabetes using Machine Learning. The model is trained using the Pima Indians Diabetes Dataset and Logistic Regression.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

---

## 🤖 Machine Learning Algorithm
- Logistic Regression

---

## 📊 Features
- Data Preprocessing
- Data Visualization
- Feature Scaling
- Model Training
- Prediction
- Model Evaluation
- Model Saving using Pickle

---

## 📁 Dataset
Pima Indians Diabetes Dataset

---

## 📈 Model Accuracy
Approximately **80%**

---

## ✅ Prediction Output

Example:

Patient has Diabetes

or

Patient does NOT have Diabetes
